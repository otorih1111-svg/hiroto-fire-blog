【依頼】ブログ用キャラクター「ひろと」＆「息子（小さな相棒）」イラスト作成

## 背景
hiroto-fire-blog（40代シングルファーザーがFIREを目指す家計・投資ブログ）の記事内図解・本文で
使うキャラクターを作りたい。
SVGの図形だけで作ったキャラがイマイチだったので、イラスト系のキャラクターに変更したい。

プロフィール写真はそのまま使うので、これは「記事を読みやすくする補助キャラ」という位置づけ。

## キャラクター設定

### ひろと（父）
- 40代男性、丸顔、短髪（白髪少し混じり）
- メガネなし
- 服はブランドカラーの緑系（#7CB089 or #1F4D32）のシャツ・パーカーなど
- ゆるい・親しみやすいタッチ（かわいい系・脱力系のイラスト）

### 息子（小さな相棒）
- 10歳の男の子
- 性格：人懐っこい、負けず嫌い、意外と頑張り屋。怒られるとすぐ泣く
- サッカーをやっている（スポーティーな雰囲気もあると◎）
- 見た目：女の子っぽい可愛らしい顔立ち。髪はひろとより少し長めのセンターパート
- 服装：パーカーをよく着ている。色味はひろとと合わせる（緑系を一部に使うなど、親子で揃ってる感）
- ひろとと同じタッチ・同じ画風で統一する（兄弟みたいに見えないよう、明確に「子ども」とわかる体型・顔立ち）

両キャラとも、初心者向けブログなので堅すぎず・クセが強すぎない雰囲気で。

## 必要なパターン

### A. ひろと（父）単独 - コア12種類
1. ニュートラル・笑顔（デフォルト）
2. 驚き（目を見開く、口が「お」の形）
3. 納得・うなずき
4. 困り・悩み（汗マーク）
5. ガッツポーズ・喜び（両手を上げる）
6. 考え中（手を顎に当てる）
7. 落ち込み・しょんぼり
8. 焦り・パニック（冷や汗）
9. ドヤ顔・自信あり（腕組み）
10. 指差し・解説（「ここ注目」のポーズ）
11. 頭を抱える（数字を見て青ざめる）
12. 小さくガッツポーズ（地味な喜び）

### B. 息子（小さな相棒）単独 - 8種類
13. ニュートラル・笑顔（デフォルト）
14. 元気いっぱい・ガッツポーズ
15. 眠そう・あくび（夜の場面用）
16. 不思議そう・「？」の表情
17. キラキラした目（喜び・期待）
18. ふくれっ面・ちょっと不満
19. 泣き顔（怒られて/悔しくて泣いている）
20. サッカーボールを持っている・プレイ中（負けず嫌い・頑張り屋な雰囲気）

### C. 2人バージョン（ひろと＋息子） - 6種類
21. 並んで笑っている（仲良し・日常）
22. ハイタッチしている（成功・うまくいった場面）
23. 一緒に画面（パソコン/スマホ）を見ている（一緒に何か学んでいる場面）
24. ひろとが息子を見守っている（やさしい・安心感のある場面）
25. 2人で頭を抱えている（うまくいかない・失敗した場面）
26. 2人で寝ている/おやすみ（夜・癒し系の場面）

## 技術仕様
- 各パターンを個別の画像ファイル（PNG、背景透過）として出力
- サイズは正方形（例: 512x512）。2人バージョンは横長（例: 800x512）でもOK
- 背景は透過にする（記事の背景色#F7FBF8や#EAF4EDの上に乗せるため）
- ファイル名は英語で内容がわかるように
  - ひろと: hiroto-neutral.png / hiroto-surprised.png / hiroto-nod.png / hiroto-worried.png /
    hiroto-guts.png / hiroto-thinking.png / hiroto-disappointed.png / hiroto-panic.png /
    hiroto-confident.png / hiroto-pointing.png / hiroto-facepalm.png / hiroto-small-guts.png
  - 息子: son-neutral.png / son-energetic.png / son-sleepy.png / son-confused.png /
    son-sparkle.png / son-pouting.png / son-crying.png / son-soccer.png
  - 2人: duo-laughing.png / duo-hightouch.png / duo-screen.png / duo-watching.png /
    duo-facepalm.png / duo-sleeping.png

## ゴール
記事の内容・感情に応じてキャラクターを差し込み、初心者にも親しみやすく・楽しく読める記事にする。
まずはA（ひろと12種類）から優先して進めて、B・Cは可能な範囲で対応する想定。
