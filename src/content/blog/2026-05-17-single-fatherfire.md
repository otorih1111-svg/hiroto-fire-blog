# シングル父のFIRE設計：今の資産状況を正直に書く

かっこ悪いけど、書く。
数字を隠したままFIREを語っても、誰の役にも立たないから。

---

## 「FIREを目指してます」と言いながら、現実から目を逸らしていた

FIREという言葉を知ったのは2年ほど前だった。

経済的自由。早期リタイア。
なんとなく「そういう生き方があるんだな」と思って、本を1冊買った。読んだ。ふーん、と思った。それで終わった。

本当に「やろう」と思ったのは、離婚後に息子と2人になってからだ。

当時の俺は、自分の口座に何円あるかも把握していなかった。
妻に任せきりにしていた。

離婚して初めて通帳を全部並べて、現実と向き合った。
貯金はほぼゼロ。毎月いくら使っているかも分からない。「知らなかった」んじゃなかった。見たくなかっただけだった。

あれから2年。
副業を始めて3ヶ月が経った。収益はまだほぼゼロだ。

でも今日は、そこじゃなくて「資産の現在地」を正直に書く。

---

## 今の数字、全部出す

読者がいる前提で数字を公開するのは、正直怖い。
でも、曖昧にしたまま「FIREを目指してます」と言い続けるのは違う気がした。

**現在の資産状況（2024年時点）**

| 項目 | 金額（概算） |
|---|---|
| 普通預金 | 約80万円 |
| 積立NISA | 約45万円（2年継続） |
| iDeCo | 約28万円（2年継続） |
| その他（株式） | 約12万円 |
| **合計** | **約165万円** |

これが45歳・シングル父の現在地だ。

ちなみに、離婚直後は貯金ゼロだった。
ここ2年で積み上げてきた数字ではあるけれど、FIREには程遠い。

---

## 毎月のお金の流れ、全部書く

**月収（手取り）：約30万円**

会社員の給与だけ。副業収益はほぼ含まれない。

**月の支出（概算）**

| 項目 | 金額 |
|---|---|
| 家賃 | 8.5万円（2LDK・千葉） |
| 食費 | 4万円（外食含む） |
| 光熱費・通信費 | 2万円 |
| 息子の習い事・教育費 | 2.5万円 |
| 保険 | 1万円 |
| 交通費・日用品など | 1.5万円 |
| **合計** | **約19.5万円** |

残り約10万円を、毎月以下に回している。

- 積立NISA：3万円
- iDeCo：2.3万円（上限）
- 普通預金（緊急資金）：3万円
- 副業の学習・ツール費用：1万円〜

これが今の家計の全体像だ。

「節約上手だね」とは言われない。
でも離婚直後の貯金ゼロと比べたら、少しずつ前に進んでいる。

---

## FIRE達成には、あと何年かかるのか

FIREに必要な資産の目安は「年間支出の25倍」と言われている。
俺の場合、年間支出は約240万円。つまり目標資産は**6,000万円**だ。

今の資産は165万円。

……そう。5,835万円足りない。

毎月の積立（投資+貯蓄）が約8.3万円、年利5%で運用し続けたとして——シミュレーターで計算してみると、約**20年**かかる計算になった。

65歳。息子は30歳。

「遅すぎる」と思う人もいるだろう。
俺も最初はそう思った。でも少し立ち止まって考えた。

今から何もしなかったら、65歳はもっと厳しい。
副業の収益が月5万円でも上乗せされれば、年数は大きく縮まる。
今の投資習慣を続ければ、複利は確実に働き始める。

数字を見るのが怖かったのは、「現実と向き合う怖さ」だったと今は思う。
見てしまえば、動ける。動けば、変わる。

---

## 「まだ遅くない」じゃなくて「今日から変わる」という話

正直に言う。
「45歳からFIRE目指してます」と言うと、笑う人もいると思う。

でも俺がFIREを目指す理由は、早期リタイアしたいからだけじゃない。

息子が何かに挑戦したいと言ったとき、「お金がないから無理だ」と答えたくない。ただ、それだけだ。

そのために今やっていること、全部書いておく。

- 毎月の収支を手帳に書く（アプリより手書きが続く）
- 積立NISAとiDeCoを自動引き落としにして「触れない仕組み」を作った
- 副業は今のところ収益ゼロだが、月1万円以内のコストで続けている
- 半年に一度、資産残高を確認して記録している

地味だ。本当に地味だ。
でも「見ない」を続けた過去の自分より、確実にマシだと思っている。

---

あなたは今、自分の資産状況を正確に言えるだろうか。

通帳、投資口座、保険の解約返戻金。全部足したら、いくらになるか。

数字を出すのが怖い気持ち、俺には分かる。
でも見てしまえば、次のステップが見えてくる。

まず今日、通帳を一冊だけ開いてみることから始めてほしい。
ゴールは遠くていい。「見ること」が先だ。
